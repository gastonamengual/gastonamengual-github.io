Mandelbrot Set

Miscellanea Mon 14 September 2020

Notebook written by Gastón Amengual.

The Mandelbrot set is the set of complex numbers \(c\) for which the function \(F(z) = z^2 + c\) does not diverge when iterated from \(z = 0\).

\(c = a + bi\), where \(a\) is the real part and \(b\) is the imaginary part.

A point (x,y) will belong to the Mandelbrot Set if and only if the modulus or absolute value of c is never larger than 2, no matter the number of iterations.

def mandelbrot(r, im, max_iter):
    c = complex(r,im)

    z = 0.0j
    for i in range(max_iter):
        z = z**2 + c
        if abs(z) > 2:
            return i

    return max_iter

rows = 2000
columns = 2000
max_iter = 100

x = np.linspace(-2,1,rows)
y = np.linspace(-1,1,columns)
result = np.zeros([rows,columns])

for i,real in enumerate(x):
    for j,imag in enumerate(y):
        result[i,j] = mandelbrot(real, imag, max_iter)
